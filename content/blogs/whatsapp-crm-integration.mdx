## How to Set Up WhatsApp CRM Integration

1. **Choose your integration path** — Identify your CRM platform. Use a pre-built connector for Salesforce, HubSpot, Zoho, Freshdesk, LeadSquared, Sell.do — or the REST API for custom CRMs.
2. **Confirm API credentials are ready** — Ensure WhatsApp onboarding is complete and your GCM API key and Phone Number ID are available.
3. **Install or configure the connector** — Install the GCM app from your CRM's marketplace and authenticate with your API key. No developer required.
4. **Map data fields** — Configure which CRM fields correspond to WhatsApp message variables — e.g. MobilePhone → recipient number.
5. **Define trigger conditions** — Set the CRM automation rules that fire WhatsApp messages — lead qualification, payment reminders, and more.
6. **Configure inbound webhook** — Post inbound WhatsApp events (message, button tap, delivery status) back to your CRM in real time.
7. **Test end-to-end** — Create a test lead, verify the WhatsApp fires, reply from your phone, confirm the CRM record updates.
